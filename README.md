This project presents an analysis of car sales data using Power BI.  
It includes KPIs, sales comparisons, price analysis, and dealer-wise performance insights.

 Key Features
- MTD, YTD, PYTD sales and KPI comparisons  
- Price analysis by Company, Model, Engine, Body Style, Color & Transmission  
- Dealer analysis (Region, Dealer Name, Dealer No)  
- Trend analysis using Date Week & Year fields  

Key Insights
- Sales performance by time periods (MTD/YTD/PYTD)   
- Cars sold difference across brands and models  
- Fuel/engine type impact on pricing and demand  

 
 Tools Used
- Power BI  
- Excel Dataset
